# Chronos Forecasting

Forecasting using https://huggingface.co/amazon/chronos-t5-large

## Install & Run

Installed and run with `Python 3.10.11` and `Poetry (version 1.8.2)`:

- `poetry install`
- `poetry shell`
- `python forecasting.py`
